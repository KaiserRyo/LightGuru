===================================================
  LightGuru
===================================================

Copyright 2013 Gael Jaffrain

LightGuru is a lightmeter application for BlackBerry 10. 
It measures the ambiant light and provide help to the photographer to set the correct exposure settings on his camera.

Release Notes
=============
1.0.0.1		Initial Release. Minimum OS: BlackBerry 10.1
1.1.0.1		UI improvements, lock feature bug fix, EV toggle feature added, improved orientation support. Minimum OS: BlackBerry 10.1.
